# mp-dl
Инструмент для загрузки и распаковки Minecraft сборок по ссылке.

# Дорожная карта
## v1
- Поддержка форматов: `.zip`, `.7z`, `.tar.gz`, `.tar.xz`
- Наличие TUI и CLI
- Выбор пути распаковки (в TUI) с помощью [системного интерфейса](https://github.com/PolyMeilex/rfd)

## v2
- Поддержка дополнительных форматов: `.mrpack` (и возможно другие)
- Вывод содержимого архива в виде дерева
- Частичное извлечение архива (только выбранные файлы/папки)

## Возможные перспективы
- Создание библиотеки для работы со сборками (`.mrpack` и прочие)
- Создание отдельного инструмента для запаковки сборок в `.zip`, `.7z`, `.tar.gz`, `.tar.xz`, `.mrpack`
